폴더 생성 (mkdir = make directory)
mkdir Moon

생성한 Moon 폴더로 이동 (cd = change directory)
cd Moon

파일 생성과 동시에 파일에 글자를 적고 싶을 때 echo
echo file2 > file2.txt

파일만 생성하고 싶을 때: touch
touch file1.txt

이미 생성된 파일에 내용을 추가하고 싶을때
echo add >> file2.txt

Staging Area에 파일추가
git add .

Staging Area 상태를 확인하고 싶을 때
git status

커밋 메세지를 한 줄만 적고 싶을때
git commit -m "aaaa"

커밋 메세지를 여러 줄 적고 싶을때
git commit

2. 너무 가끔 커밋: 수십 수백 개의 파일로 만들어서 한 번 커밋
